This is a simple project that we started to get into app and web development and as a fun side quest.
It is basically a hazard reporting app when citizens can report hazards in their neighbourhood to our app.
The authorites can view the hazards and can update the status of the reports from their admin dashboard.
This has basically 2 parts:
  1. App: It is used by users to report any hazards.
  2. WebApp: It is a dashboard used to view the reports made by citizens. Authorities can edit the status of the hazards while the citizens can only view them.
